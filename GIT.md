# 1. Clone repo
```commandline
git clone git@github.com:mykytapavlov/python-beginner.git
```

# 2. Status
```commandline
git status
```

# 3. Commit
```commandline
git add <file name>
git commit -m "commit msg"
git push


git log -1 # to see last commit in local history
```

# 4. Create Branch
```commandline
git pull
git checkout -b <branch_name>
git branch
```
